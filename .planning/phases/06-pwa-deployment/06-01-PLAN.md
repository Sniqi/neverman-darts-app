RESEARCH.md suggests `src/lib/ReloadPrompt.svelte` and importing via `$lib`. A Svelte component with browser-mode tests MUST live under `src/ui/`. Therefore place the component at `src/ui/pwa/ReloadPrompt.svelte` and its test at `src/ui/pwa/ReloadPrompt.test.ts` so the existing browser project picks it up. </placement_decision>
